Privacy

Last updated: 20 July 2026

Video Subtitle Overlay is a static, open-source browser app. It has no application server, account database, analytics, advertising, or telemetry.

Google access is read-only. The app requests the drive.readonly scope through Google Identity Services. It can list Drive file names and folders and download the video you choose, but it cannot upload, edit, share, or delete Drive files and does not request Gmail access.

Data the app handles

  • Local files: videos, subtitles, and metadata you select remain inside your browser tab and are not uploaded by this app.
  • Public or direct links: your browser connects to YouTube, Google Drive, or the URL host you supplied.
  • Private Google Drive files: Google returns folder and file names, IDs, MIME types, sizes, and modification times while you browse. The app then downloads the content of the video you choose.
  • Deployment configuration: the public OAuth client ID may be saved in local storage. No API key or client secret belongs in this app.
  • Preferences: overlay sizes, visibility, background style, and positions are saved in local storage.

How Google user data is used

The selected Drive file is downloaded directly from the Google Drive API into a temporary in-memory browser blob solely so the app can play it with subtitles, metadata, and a timecode overlay. The app does not sell, share, transfer, train on, analyze, or use Google user data for advertising.

Storage and retention

  • The Google access token is held only in JavaScript memory. It is never written to local storage, a cookie, a file, or an application server.
  • The access token disappears when the page reloads or closes, expires after Google's short lifetime, and is revoked when you choose Disconnect Google Drive.
  • A buffered Drive video remains in memory only until you replace it, disconnect Drive, reload, or close the tab.
  • The public OAuth client ID and UI preferences remain in local storage until you clear this site's browser data.

Disclosure and third parties

This app has no backend to receive your data. Google processes authorization and Drive API requests under the Google Privacy Policy. YouTube and any direct media host process requests made to their services.

Your controls

Use Menu → Disconnect Google Drive to revoke the current Google grant and remove any private buffered video from the player. You can also revoke the app from your Google Account's third-party access page, close the tab, or clear this site's browser storage.

Google API Services policy

Video Subtitle Overlay's use and transfer of information received from Google APIs adheres to the Google API Services User Data Policy, including its Limited Use requirements.

Contact

Questions or deletion concerns can be raised through the repository's GitHub issue tracker. Because the app has no server-side account or storage, there is no server-held user record for the project to delete.
