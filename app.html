🎬 Video + Subs + Metadata

Click Pick folder and choose a folder containing your videos.

For each video, files with the same base name are auto-loaded:
• talk.mp4 — the video
• talk.vtt — subtitles
• talk.json or talk.txt — metadata overlay

No metadata file yet? Click ✎ New metadata to build one and download the .json.

You can also drag & drop files (or a whole folder) anywhere on the page.

Share via Chrome tab (with "Share tab audio") — subtitles and metadata are drawn in the tab, so they show in the shared view.
